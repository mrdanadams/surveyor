Surveyor

Test your responsive designs at all breakpoints, side by side.

Enter a URL & width breakpoints to test. You'll see them all at once, side by side. Try it and watch your CSS bugs disappear.

Surveyor is meant to make testing your responsive sites easier so you can find the problems that inevitably come up when doing any non-trivial design for many screen sizes and devices.
